What isFxCodeGeneratedEA

FxCodeGeneratedEA เป็นการเว็บไซต์สร้างโค้ดการซื้อขายอัตราการแลกเปลี่ยนระหว่างประเทศ (Foreign Exchange Market) โดยช่วยให้คุณสามารถสร้างกลยุทธ์ หรือกระบวนการคิดของคุณเองได้อย่างง่ายดาย เพียงแค่เลือก อินดิเคเตอร์ (Indicator) ที่คุณต้องการและทำการกำหนดค่าต่างๆเท่านั้นเอง

Trading algorithms

กลยุทธ์ในการเทรด หรือแนวคิดในการเทรด เว็บไซต์ของเรามีอินดิเคเตอร์ (Indicator) ถึง14ตัว โดยแบ่งเป็นโหมดเทรน (Trend) 7ตัว และออสซิลเลเตอร์ (oscillator) 7ตัว ที่สามารถใช้บนแพลตฟอร์ม MT4 โดยสามารถเปิดและปิด Position ได้ตามตำแหน่งที่ผู้ใช้ต้องการ

Layer system money manager

EA ที่ท่านได้กำหนดกลยุทธ์ในการเทรด หรือเงื่อนไขต่างๆจากเว็บไซร์ของเรา ทุก EA จะมีการบริหารเงิน (Money Manager) เป็นแบบชั้น (Layer) คือรูปแบบที่เราจะตัดสินใจใน 1 โมเดลหรือกลยุทธ์นั้นๆด้วยกลยุทธ์ Signal จากทางเทคนิคของท่าน รุปแบบนี้จึงจำกัดความเสี่ยงไว้ทั้งหมดแล้วที่ตัว Layer ซึ่งเรากำหนดไว้ 4 Layer และท่านจะไม่มีทางเทรดเกิน Layerที่เราออกแบบไว้ ส่วนสำหรับ Stoploss ในแต่ล่ะ Layer นั้นอยู่ที่ท่านจะเป็นคนกำหนด และความเสี่ยงที่เรายอมรับได้
